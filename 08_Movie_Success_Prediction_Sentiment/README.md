# Movie Success Prediction and Sentiment Study

## Objective
Predict movie success using IMDB/Kaggle data, and analyze sentiment of viewer reviews.

## Tools Used
Python (Pandas, NLTK, TextBlob, Scikit-learn, Matplotlib)

## Dataset Suggestions
- TMDB 5000 Movies (Kaggle): https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
- IMDB Dataset (Kaggle): https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

## Steps Involved
1. Load IMDB/TMDB movie dataset with ratings and financial data
2. Apply TextBlob/VADER sentiment analysis on user reviews
3. Engineer features: budget, genre encoding, cast popularity score, sentiment scores
4. Train regression model to predict box office revenue
5. Analyze genre-wise sentiment trends and correlations
6. Generate sentiment visualizations and predictive model summary

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
