# Retail Business Performance & Profitability Analysis

## Objective
Analyze transactional retail data to uncover profit-draining categories, optimize inventory turnover, and identify seasonal product behavior.

## Tools Used
Python (Pandas, Seaborn, Matplotlib, SQLite)

## Dataset Suggestions
- Superstore Dataset (Kaggle): https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
- Retail Data Analytics (Kaggle): https://www.kaggle.com/datasets/manjeetsingh/retaildataset

## Steps Involved
1. Load the Superstore retail dataset and perform initial cleaning (handle missing values, correct data types, remove duplicates)
2. Use SQLite to calculate profit margins by category and sub-category with SQL queries
3. Compute correlation between shipping/inventory days and profitability using Pandas
4. Build visualizations: profit margin heatmaps, seasonal trend line charts, category-wise bar charts
5. Derive strategic recommendations for slow-moving and overstocked items
6. Generate final dashboard-style visualization combining all key insights

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
