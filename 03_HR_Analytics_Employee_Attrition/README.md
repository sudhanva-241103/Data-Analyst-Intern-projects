# HR Analytics - Predict Employee Attrition

## Objective
Use analytics to understand the main causes of employee resignation and predict future attrition.

## Tools Used
Python (Pandas, Seaborn, Scikit-learn, SHAP, Matplotlib)

## Dataset Suggestions
- IBM HR Analytics (Kaggle): https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset
- HR Employee Attrition (Kaggle): https://www.kaggle.com/datasets/patelprashant/employee-attrition

## Steps Involved
1. Load IBM HR Analytics dataset and perform EDA (department-wise attrition, salary distributions, promotion analysis)
2. Encode categorical variables and scale numerical features
3. Build Logistic Regression and Decision Tree classification models
4. Evaluate using accuracy, precision, recall, F1-score, and confusion matrix
5. Perform SHAP value analysis to explain model predictions
6. Generate attrition prevention recommendations based on findings

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
