# E-commerce Return Rate Reduction Analysis

## Objective
Identify why customers return products and how return rates vary by category, geography, and marketing channel.

## Tools Used
Python (Pandas, Scikit-learn, Matplotlib, Seaborn)

## Dataset Suggestions
- E-Commerce Shipping Dataset (Kaggle): https://www.kaggle.com/datasets/prachi13/customer-analytics
- Brazilian E-Commerce (Kaggle): https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Steps Involved
1. Load and clean the e-commerce orders and returns dataset
2. Analyze return percentage per category, supplier, and region
3. Engineer features: price range, discount percentage, delivery time, customer history
4. Train logistic regression model to predict probability of return
5. Create return risk score dashboard with category and region breakdowns
6. Export CSV of high-risk products with actionable recommendations

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
