# YouTube Trending Video Analytics

## Objective
Uncover patterns in trending videos by analyzing YouTube datasets across regions.

## Tools Used
Python (Pandas, Matplotlib, Seaborn, TextBlob, SQLite)

## Dataset Suggestions
- YouTube Trending Videos (Kaggle): https://www.kaggle.com/datasets/datasnaek/youtube-new
- YouTube Statistics (Kaggle): https://www.kaggle.com/datasets/nelgiriyewithana/global-youtube-statistics-2023

## Steps Involved
1. Load and clean YouTube trending datasets, standardizing columns across regions
2. Perform sentiment analysis on video titles and tags using TextBlob
3. Use SQLite to rank categories by average views, likes, and engagement ratios
4. Create time-series visualizations for trending duration and publish timing
5. Build region-wise comparison visuals for category preferences
6. Compile findings into a data storytelling narrative

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
