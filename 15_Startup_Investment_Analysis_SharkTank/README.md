# Startup Investment Analysis (Shark Tank Data)

## Objective
Analyze startup investment trends using Shark Tank India/US datasets.

## Tools Used
Python (Pandas, Matplotlib, Seaborn)

## Dataset Suggestions
- Shark Tank India Dataset (Kaggle): https://www.kaggle.com/datasets/thirumani/shark-tank-india
- Shark Tank US Dataset (Kaggle): https://www.kaggle.com/datasets/ulrikthygepedersen/shark-tank-companies

## Steps Involved
1. Load and clean Shark Tank dataset (organize by domain, funding amount, deal status)
2. Analyze deal success rates by industry, ask amount, and equity offered
3. Profile investor patterns: which sharks invest in which domains
4. Examine founder demographics and their correlation with deal success
5. Create visualizations for industry trends, funding distributions, and deal outcomes
6. Generate founder success pattern summary with key takeaways

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
