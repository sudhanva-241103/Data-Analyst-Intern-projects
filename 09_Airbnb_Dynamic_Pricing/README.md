# Airbnb Dynamic Pricing Recommendation Engine

## Objective
Analyze historical Airbnb data to suggest optimal pricing based on location, season, and listing quality.

## Tools Used
Python (Pandas, Scikit-learn, Matplotlib, Seaborn)

## Dataset Suggestions
- Airbnb Open Data (Inside Airbnb): http://insideairbnb.com/get-the-data/
- Airbnb NYC Dataset (Kaggle): https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data

## Steps Involved
1. Load and clean Airbnb listings dataset (handle missing values, parse amenities)
2. Analyze pricing distribution by city, property type, and review scores
3. Engineer features: amenity count, distance to center, review sentiment, season
4. Train regression model (Random Forest) to predict optimal price
5. Build pricing recommendation function with adjustable parameters
6. Create dashboard visualizations with pricing insights and suggestions

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
