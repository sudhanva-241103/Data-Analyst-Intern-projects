# Global CO2 Emissions Tracker by Sector

## Objective
Build a dashboard to track carbon emissions from energy, transport, and industry sectors across countries.

## Tools Used
Python (Pandas, Matplotlib, Seaborn, Plotly-style charts)

## Dataset Suggestions
- Our World in Data CO2 Dataset: https://github.com/owid/co2-data
- Global Carbon Project: https://www.kaggle.com/datasets/thedevastator/global-fossil-co2-emissions-by-country-2002-2022

## Steps Involved
1. Load multi-year global emissions dataset (Our World in Data / OWID)
2. Clean and prepare per capita and per GDP emissions metrics
3. Create sector-wise breakdown visualizations (Energy, Transport, Industry)
4. Build country comparison charts and ranking tables
5. Analyze trends over time for top 10 emitting countries
6. Generate PDF policy brief summarizing key findings on top polluters

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
