# Financial KPI Analysis for a Startup

## Objective
Analyze monthly revenue, burn rate, CAC, LTV, and run rate for an early-stage startup.

## Tools Used
Python (Pandas, Matplotlib, Seaborn)

## Dataset Suggestions
- Simulated startup financial data (generated in code)
- Startup Investments (Crunchbase-style): https://www.kaggle.com/datasets/arindam235/startup-investments-crunchbase

## Steps Involved
1. Generate or load startup financials: monthly expenses, revenue, and customer base data
2. Compute key KPIs: CAC, LTV, LTV:CAC ratio, burn rate, and runway
3. Perform cohort analysis grouping customers by acquisition month
4. Build trend visualizations with KPI indicators (green/red thresholds)
5. Create a financial health dashboard combining all metrics
6. Export Excel-style summary template with formulas

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
