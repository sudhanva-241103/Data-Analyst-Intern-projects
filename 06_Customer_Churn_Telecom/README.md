# Customer Churn Analysis for Telecom Industry

## Objective
Predict churn and derive actionable strategies to retain users in a highly competitive telecom environment.

## Tools Used
Python (Pandas, Scikit-learn, SHAP, Matplotlib, Seaborn)

## Dataset Suggestions
- Telco Customer Churn (Kaggle): https://www.kaggle.com/datasets/blastchar/telco-customer-churn
- IBM Telecom Churn (Kaggle): https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset

## Steps Involved
1. Load telecom dataset and perform data aggregation (call duration, complaints, recharge frequency)
2. Perform EDA: churn rate by tenure, contract type, payment method
3. Handle class imbalance using SMOTE or class weights
4. Build binary classification model (Random Forest) for churn prediction
5. Use SHAP for model explainability and feature importance
6. Create customer segments: At Risk, Loyal, Dormant with retention recommendations

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
