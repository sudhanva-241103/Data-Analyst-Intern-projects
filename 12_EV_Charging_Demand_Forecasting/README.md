# Electric Vehicle Charging Demand Forecasting

## Objective
Forecast the demand at EV charging stations based on weather, time, and traffic.

## Tools Used
Python (Pandas, Scikit-learn, Matplotlib, Seaborn, Statsmodels)

## Dataset Suggestions
- EV Charging Data (simulated in code or use ACN-Data): https://ev.caltech.edu/dataset
- Electric Vehicle Population (Kaggle): https://www.kaggle.com/datasets/ratikkakkar/electric-vehicle-population-data

## Steps Involved
1. Generate or load EV charging station usage data with timestamps
2. Merge with weather data (temperature, precipitation, wind)
3. Perform time-series decomposition (trend, seasonality, residual)
4. Build forecasting model using Statsmodels (exponential smoothing or linear regression with time features)
5. Create demand heatmaps by hour and day of week
6. Generate charging optimization strategy recommendations

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
