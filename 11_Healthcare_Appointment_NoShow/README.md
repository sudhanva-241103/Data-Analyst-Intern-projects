# Healthcare Appointment No-Show Prediction

## Objective
Predict whether patients will miss their appointments and optimize scheduling.

## Tools Used
Python (Pandas, Scikit-learn, Matplotlib, Seaborn)

## Dataset Suggestions
- Medical Appointment No-Shows (Kaggle): https://www.kaggle.com/datasets/joniarroba/noshowappointments
- Healthcare Dataset (Kaggle): https://www.kaggle.com/datasets/prasad22/healthcare-dataset

## Steps Involved
1. Load and clean the medical appointment no-show dataset
2. Perform EDA: no-show rates by age group, weekday, SMS reminder status, neighborhood
3. Engineer features: wait time (days between scheduling and appointment), age groups
4. Train Decision Tree and Random Forest models to predict no-shows
5. Evaluate with accuracy, precision, recall, and confusion matrix
6. Generate optimization recommendations for appointment scheduling

## How to Run
1. Place your dataset file in this folder (or use the sample data generated by the script)
2. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Run: `python analysis.py`
4. Check generated PNG dashboard and any CSV outputs

## Deliverables
- `analysis.py` — Complete Python analysis code
- `report.pdf` — 1-2 page project report
- Dashboard visualization (PNG, generated when you run the code)
- Any CSV/Excel outputs (generated when you run the code)
